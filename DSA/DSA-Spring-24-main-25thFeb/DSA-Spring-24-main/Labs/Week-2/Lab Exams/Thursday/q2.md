# Addition Problem!

<!-- # Problem Statement -->

<!----------------------->

Mr Rastogi, always up for a twist, handed you two numbers of lengths `n` and `m`. He wants you to represent them as linked lists, with each digit as a node. He also wants you to perform addition operation using the linked lists and store the result in a new linked list. Are you up for this task?

### Input Format

<!----------------------->

- The first line of each test case will contain two integer `n` and `m` representing the lengths of the two integer numbers.
- Next two lines will contain the two numbers of respective lengths.

### Output Format

<!----------------------->

You need to print the final linked list as the output.

## Constraints

<!----------------------->

<!-- ~ 1 \leq n, m \leq 100000~ -->

$1 \leq n, m \leq 100000$

The value in the linked list is always non-negative.

## Sample Test Case

<!----------------------->

### Input 1

```
4 5
1234
11111
```

### Output 1

```
12345
```

### Input 2

```
3 1
999
1
```

### Output 2

```
1000
```
